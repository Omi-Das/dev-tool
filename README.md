...)A little Description:

                     This is a simple dynamic website built with React, where all functionalities are implemented inside the application. The data is loaded dynamically from JSON. When the "Buy Now" button is clicked, the selected item is added to the cart, and the cart updates dynamically. If an item is removed, it is deleted from the cart instantly. When the "Proceed to Checkout" button is clicked, all items in the cart are cleared, and a message saying "Your cart is empty" is displayed.
